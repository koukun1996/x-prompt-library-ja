# CLAUDE.md

このファイルは、AI アシスタント（Claude など）がこのリポジトリで作業する際のガイドです。

## プロジェクト概要

**x-prompt-library-ja** は、日本語のプロンプトライブラリです。プロジェクトは初期段階にあり、基本的なリポジトリ構造のみが存在します。

## リポジトリ構成

```
x-prompt-library-ja/
├── README.md          # プロジェクト説明
└── CLAUDE.md          # AI アシスタント向けガイド（本ファイル）
```

## 開発ワークフロー

### ブランチ戦略

- メインブランチ: `main`
- 機能ブランチは `main` から作成する

### コミット規約

- コミットメッセージは変更内容を簡潔に記述する
- 日本語または英語で記述可

## コーディング規約

- ファイル名は英数字とハイフンを使用（例: `my-prompt.md`）
- テキストファイルのエンコーディングは UTF-8
- 改行コードは LF

## 注意事項

- このプロジェクトは初期段階です。構成やルールは今後変更される可能性があります
- 新しいファイルやディレクトリを追加する際は、このファイルの「リポジトリ構成」セクションも更新してください
